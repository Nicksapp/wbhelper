# WBHelper

> 工作台辅助 cli 工具

## Getting Started

```bash
$ npm i wbhelper -g
$ wbhelper -h
```

## Function

- nei rpc 接口拉取代码生成
- 七鱼消息主动模拟
- 模块代码自动生成
